# ecommerce-sql-analysis-
# E-Commerce Business Intelligence & Analytics Engine

## Project Overview
This project showcases a series of structured SQL queries designed to extract critical business metrics from transactional database models. The goal is to analyze customer purchasing behavior, isolate high-value revenue regions, and identify performance anomalies using advanced data retrieval techniques.

## Core SQL Techniques Applied
* **Multi-Table Joins & Aliasing**: Extracting cross-table values using `LEFT JOIN` structures.
* **Data Aggregation**: Summarizing large datasets using `COUNT`, `SUM`, and `AVG` functions.
* **Strategic Filtering**: Consolidating records with group constraints using `GROUP BY`.
* **Advanced Subqueries**: Implementing isolated single-table lookups and row-by-row **Correlated Subqueries**.

## Key Analytical Questions Solved
1. What are the store's top 5 highest-valued individual transactions?
2. Which geographic regions generate the highest total revenue and order volume?
3. Which orders sit above the company's baseline average transaction value?
4. Which customer records contain cross-categorical purchases (specifically Electronics)?
5. Which professionals exceed the internal average earnings metrics within their specific departments?
